Hello! This is the website I am making after finishing the JS course.

To be able to properly learn JavaScript, I needed to know the syntax (vocabulary) of the language.
The course I watched covered this part.

JavaScript is mainly used in websites and the bricks and paint for a website is HTML and CSS.
They make the visuals (buttons, text, lists, etc.) you see on a webpage.

After acquiring the tools to build a website, we can start making one.
But first, we need to make some planning.
For planning, DOM (Document Object Model) is used.
The website planning is made visually with simpler methods using shapes and boxes.

Learning and using HTML, CSS, and JavaScript is easy.
Before starting to build a website, proper planning is needed.
DOM can be used to make that planning.
